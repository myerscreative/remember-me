## Workflow Comparison

### Setting a Contact Date

**Option A:**
1. Click Edit button in header
2. Click "Set Next Contact Date"
3. Choose date
4. Click Save

**Option B:**
1. Click Edit button in header
2. Click "Schedule Next Contact" in empty card
3. Choose date and add reason (optional)
4. Click Save

### Marking as Contacted

**Option A:**
Not available - must manually update date

**Option B:**
1. Click "Mark as Contacted" button
2. Done! (Auto-suggests next date)

### Rescheduling

**Option A:**
1. Click date (in edit mode)
2. Change date
3. Save

**Option B:**
1. Click "+1 Week" or "+1 Month" (in edit mode)
2. Done! (Auto-saves)

## Recommended Usage

### Start with Option B
Most users should start with **Option B** because:
- More discoverable for new users
- Teaches the nurturing habit
- Provides more guidance
- Shows all available features

### Switch to Option A when:
- You're comfortable with the system
- You want a cleaner interface
- You have your own nurturing workflow
- You check contacts regularly anyway
- You want more screen space for other info

### Stay with Option B if:
- Nurturing relationships is your primary use case
- You like visual reminders
- You want one-click actions
- You manage many contacts actively
- You're using this as a full CRM
