# PWA Icons

To complete the PWA setup, add the following icon files to this directory:

## Required Icons

1. **icon-192.png** - 192x192 pixels
   - Used for app icon on Android home screen
   - Should be a PNG with transparent or solid background

2. **icon-512.png** - 512x512 pixels
   - Used for high-resolution displays
   - Should be a PNG with transparent or solid background

## Icon Design Guidelines

- Use a simple, recognizable symbol
- Ensure good contrast for visibility
- Consider using the brand colors:
  - Primary: Purple (#8b5cf6)
  - Secondary: Blue (teal-ish)
  - Accent: Teal/turquoise

## Suggested Icon Concept

For "ReMember Me", consider:
- A stylized brain with connection nodes
- Interlocking circles representing people
- A memory/bookmark icon
- The letters "RM" in a creative design

## Quick Generation

You can use tools like:
- [Favicon.io](https://favicon.io/) - Generate from text or image
- [RealFaviconGenerator](https://realfavicongenerator.net/) - Complete PWA icon set
- Design in Figma/Sketch and export at required sizes

## Testing

After adding icons:
1. Clear browser cache
2. Visit the site on mobile
3. Add to home screen
4. Check that the correct icon appears
